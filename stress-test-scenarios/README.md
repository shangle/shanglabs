# Stress Test Scenarios

## Purpose
Simulate baseline, adverse, and severely adverse scenarios to test bank capital adequacy.

## Features
- Input baseline metrics (capital ratio, NPL, NIM, ROA, efficiency)
- Configure stress shocks (GDP change, loan growth, NPL spike, interest rate rise)
- 3 scenarios: Baseline, Adverse, Severely Adverse
- Capital adequacy charts (Tier 1, LCR)
- Asset quality analysis (NPL trends)
- Profitability metrics (ROA, NIM)
- Scenario comparison radar chart
- PDF report export
