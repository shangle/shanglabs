Loan Portfolio Analyzer Help

What Is This Tool?

The Loan Portfolio Analyzer provides a comprehensive view of a bank's loan book. Upload portfolio data to see risk distribution, performance metrics, concentration analysis, and generate PDF reports for management or examiners.

Getting Started

  1. Upload Data -- Drop a CSV or Excel file containing your loan portfolio, or click "Load Sample Portfolio" to explore with demo data.
  2. Review Metrics -- View total loans, portfolio value, average interest rate, and default rate at a glance.
  3. Analyze Charts -- Explore risk distribution, loan type breakdown, and maturity profiles through interactive charts.
  4. Export Report -- Generate a PDF report summarizing all portfolio analytics.

Key Metrics

  • Total Loans: Number of active loans in the portfolio.
  • Portfolio Value: Sum of all outstanding loan balances.
  • Average Interest Rate: Weighted average rate across the portfolio.
  • Default Rate: Percentage of loans currently in default status.

Supported File Formats

  • CSV: Comma-separated with headers for loan ID, balance, rate, type, risk grade, maturity date, and status.
  • Excel (.xlsx, .xls): Standard spreadsheet format with the same column structure.
Tip: The sample data includes a realistic mix of commercial, residential, consumer, and agricultural loans to demonstrate all analytics features.

Risk Distribution

Loans are classified by risk grade (A through E) based on credit quality indicators. The risk distribution chart shows concentration by grade, helping identify potential portfolio weaknesses.

Note: All data processing happens in your browser. No loan data is uploaded to any server. Your portfolio information stays completely private.
