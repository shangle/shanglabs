# Quick Start Guide - ROI Calculator

## Open in Browser

Simply open `index.html` in any modern web browser:
- Chrome/Edge/Firefox/Safari
- Double-click the file, or
- Right-click → "Open with" → Your browser

## Quick Test - Simple Mode

1. Click "Start Simple"
2. Step 1: Enter Revenue: `50000`
3. Step 2: Enter Costs: `20000`
4. Step 3: Keep "6 Months" selected
5. Click "Calculate"
6. You should see:
   - ROI: 150%
   - Net Profit: $30,000
   - A doughnut chart showing the breakdown

## Quick Test - Advanced Mode

1. Click "Back" then "Start Advanced"
2. Step 1: Enter:
   - Initial Investment: `100000`
   - Analysis Period: `3 Years`
   - Annual Revenue: `50000`
   - Annual Costs: `10000`
   - Discount Rate: `10`
3. Click "Next" to proceed through the wizard
4. Step 4: Adjust sensitivity sliders to see how ROI changes
5. Step 5: Click "Calculate"
6. You should see:
   - NPV, IRR, and ROI values
   - Cash flow line chart
   - ROI comparison bar chart

## Export Functions

1. In Simple or Advanced results, click "Export CSV" to download a CSV file
2. Click "Export PDF" to download a formatted PDF report

## Technical Requirements

- JavaScript must be enabled
- Internet connection (for CDN loads on first use)
- Modern browser (last 5 years)

## Deployment

Since this is client-side only:

1. Upload all files to any web server (GitHub Pages, Netlify, Vercel, etc.)
2. Or simply open index.html locally
3. No backend required!
4. No configuration needed!

## Privacy

✓ 100% client-side
✓ No data transmitted
✓ Work offline (after initial load)
✓ Your financial data stays on your device
