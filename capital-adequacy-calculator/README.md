# Capital Adequacy Calculator

## Purpose
Calculate and monitor bank capital adequacy ratios against regulatory requirements.

## Features
- Input bank assets (risk-weighted, total)
- Input capital components (CET1, Tier 1, Total Capital)
- Configure regulatory minimums (Tier 1, Total Capital, Leverage)
- Real-time ratio calculations
- Status indicators (Well Capitalized/Below Limit)
- Capital adequacy vs requirements chart
- Historical trend chart
- Capital surplus calculation
- PDF report export
