# Liquidity Risk Dashboard

## Purpose
Monitor bank liquidity risk with real-time ratios and projections.

## Features
- LCR and NSFR monitoring
- Cash position tracking
- 30-day liquidity gap projection
- 4 charts: ratios trend, cash projection, funding sources, stress test
- Alert system for liquidity breaches
- Risk factors table with thresholds
