### About GenDigital AI Agents
The site `https://ai.gendigital.com/` lists available AI agents. These are NOT installable tools - they are AI models/services you can use.

### Recommended Approach for Better Organization:

**Option 1: Use OpenClaw's Native Capabilities**
- Already have subagents capability
- Can spawn specialized agents for different tasks
- Use `sessions_spawn` to create isolated coding sessions

**Option 2: Install ClawHub Skills**
ClawHub (clawhub.com) has pre-built skills for:
- Testing & QA
- Documentation
- Code generation
- Task management

To install skills:
```
# Browse available skills
clawhub search <topic>

# Install a skill
clawhub install <skill-name>
```

**Option 3: Use GitHub Projects**
- Create project board at github.com/shangle/shanglabs/projects
- Track tasks, bugs, features
- Link commits to issues
